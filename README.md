# installer-for-things
Installs and downloads things. You can see what in the code. By clicking on the files in the repo.

# How to use it
To use it you need some things.
1. You need at lest Python version 3.14.0
2. You need to download the modules here:
requests, bs4, tqdm.

Quick pip install command for all three: pip install requests beautifulsoup4 tqdm 
or python -m pip install requests beautifulsoup4 tqdm

Also make sure it's all up-to-date

3. You need Windows this shold not work on linux but you are free to test out
4. Make sure that all files are in the same folder

When your done with that then start menu.bat then go from there
